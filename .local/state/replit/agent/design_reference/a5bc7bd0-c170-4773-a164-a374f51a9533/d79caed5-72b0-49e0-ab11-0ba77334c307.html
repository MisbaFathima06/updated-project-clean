How to Replace Backend Files

Step 1: Keep Your Frontend Intact

Your client folder stays exactly as it is. Only replace server files.

client/ ← Keep all these files unchanged

Step 2: Replace Server Files

Replace these specific files in your server directory:

• server/services/*.ts ← All service files
• server/routes/*.ts ← All route files
• server/index.ts ← Main server file
• server/db/schema.ts ← Database schema

Step 3: Environment Setup

Update your .env file with these variables:

DATABASE_URL=postgresql://...
IPFS_API_URL=https://ipfs.infura.io:5001
BLOCKCHAIN_RPC_URL=https://...
EMERGENCY_CONTACT_1=+911
JWT_SECRET=your_secret_here

✅ Result: Fixed Backend

Your complaint submission will work perfectly with the existing frontend!
